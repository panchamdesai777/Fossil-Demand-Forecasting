PROBLEM STATEMENT

Fossil is a leading global fashion designer and manufacturer, famous for its watches, handbags, and other accessories. As a retailer of a wide variety of products, both overproduction and underproduction can pose problems for the business. Therefore predicting demand as accurately as possible is critical to Fossil. Because of the normal production lead time, Fossil needs accurate forecasts months in advance.

The demand for Fossil watches is highly dynamic and influenced by seasonal trends, marketing campaigns, and regional preferences. Predicting demand with high accuracy is crucial to ensure:

  • Reduced inventory holding costs
  • Minimized overstock and stockouts
  • Improved customer satisfaction

Categorical Features

Feature Description

Numerical Features

Feature Description

SOLUTION OVERVIEW

To predict demand accurately for Fossil's products, a structured and systematic approach was followed, using advanced machine learning techniques:

Step 1: Data Preprocessing

Cleaned the data by removing negative values in the target variable. Performed label encoding and word2vec on categorical features to ensure they are suitable for model input.

Step 2: Feature Engineering

Created lead-lag features to assist the model in predicting the demand for SKUs by incorporating past data points to forecast future demand.

Step 3: Temporal Feature Engineering

Created an ARIMA-based feature using the 'sellout' column to capture temporal dynamics and enhance the forecasting power of time series modeling.

Step 4: Hyperparameter Optimization

Optimized model parameters using Bayesian optimization, ensuring the best-performing model configuration for demand forecasting.

Step 5: Model Evaluation and Finalization

Created a Stacking model (Combination Of XGBoost, Catboost, LightGbm) and evaluated it using K-fold cross-validation, with Mean Absolute Percentage Error (MAPE) as the primary metric to assess model accuracy and performance.
